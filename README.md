# CashFrog Site

Static GitHub Pages site for CashFrog.

## Files

- `index.html` - landing page.
- `privacy.html` - privacy policy for app store publication.
- `styles.css` - shared styles.
- `assets/hero-cashfrog.png` - landing page hero image.

## Publish

Enable GitHub Pages in repository settings:

1. Settings -> Pages.
2. Source: Deploy from a branch.
3. Branch: `main`.
4. Folder: `/root`.
