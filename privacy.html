Политика конфиденциальности

Дата обновления: 18 июня 2026 года

1. Какие данные обрабатывает приложение

CashFrog помогает хранить карты, категории кешбэка, проценты, лимиты и связанные настройки, которые пользователь добавляет в приложении.

Если пользователь выбирает импорт со скриншота, приложение отправляет выбранное изображение на OCR-сервер для распознавания текста и извлечения категорий кешбэка.

2. Как используются скриншоты

Скриншоты используются только для распознавания категорий кешбэка. Изображения обрабатываются временно во время запроса и не используются для рекламы или продажи третьим лицам.

Пользователь может не использовать импорт скриншотов и вводить категории вручную.

3. Хранение данных

Данные карт и категорий хранятся на устройстве пользователя. При использовании OCR-импорта на сервер отправляется только выбранное пользователем изображение.

4. Передача третьим лицам

CashFrog не продает пользовательские данные. Данные могут обрабатываться техническими сервисами, необходимыми для работы OCR-сервера и публикации приложения.

5. Безопасность

Разработчик принимает разумные технические меры для защиты передаваемых данных. Пользователю рекомендуется не загружать скриншоты, содержащие лишние персональные данные, если они не нужны для распознавания категорий.
